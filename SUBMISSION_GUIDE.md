### 2️⃣ **GAME LOGIC DESCRIPTION** (30 points)
**What:** Written explanation of how the game works  
**How to provide:**
- Use the template format provided in assignment
- Fill in these 6 sections:

```
AQUA RUSH - Game Logic

1. Game Start:
   - When the game loads, title screen appears with charity: water logo
   - Players can click "START GAME" button
   - Game initializes with Score: 0, Timer: 60 seconds, Combo: 1x
   - Droplets begin falling immediately
   
2. Player Actions:
   - Player clicks/taps on blue droplets: scores +10 points × combo multiplier
   - Player clicks on black droplets: loses -15 points and combo resets
   - Player clicks empty space: nothing happens
   
3. Game Logic:
   - Blue droplets spawn continuously (80% safe droplets)
   - Black droplets are obstacles to avoid (20% pollution)
   - Combo multiplier increases with consecutive safe taps:
     * 5 in a row = 2x multiplier
     * 10 in a row = 3x multiplier
     * 20+ in a row = 4x multiplier
   - Difficulty increases every 15 seconds (more droplets, faster speed)
   - Game ends when timer reaches 0
   
4. Score/Feedback:
   - Score updates in real-time (top-left, yellow text)
   - Combo counter displays current multiplier (top-center, fire emoji)
   - Timer counts down (top-right, turns red at 10 seconds)
   - Floating text appears on tap: "+10" or "-15"
   - Sound effects on collect/miss
   - "People Helped" counter tracks real impact
   
5. Win/Lose Conditions:
   - Game ends when 60-second timer reaches 0
   - No "winning" score—just personal high scores
   - End screen shows:
     * Final score and stats
     * "You helped XXX people access clean water!"
     * Ranking (compared to high scores)
   
6. Reset/Replay:
   - Player clicks "PLAY AGAIN" button
   - All variables reset to initial state
   - Score: 0, Timer: 60, Combo: 1x
   - New game begins immediately
   - High scores are saved (never erased)
```

**Use:** AQUA_RUSH_GAME_LOGIC.md as reference (it has much more detail)  
**Length:** Should be thorough but concise (500-800 words)
