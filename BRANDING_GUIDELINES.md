# charity: water Brand Guidelines for Your Game

## About charity: water

**Mission:** To bring clean, safe drinking water and sanitation to every person in the world.

**Website:** charitywater.org

**Key Values:**
- Transparency
- Action-oriented
- Community-focused
- Hope and possibility
- Impact-driven

---

## Visual Identity

### Primary Colors

**Blue (#5DBCD2 or similar)**
- Trust, water, reliability
- Use for: Main background, primary UI elements, water droplets
- Emotion: Calm, trustworthy, clean

**Yellow (#F4D03F or similar)**
- Action, hope, energy
- Use for: Calls to action, jerry cans, achievements, highlights
- Emotion: Optimistic, energetic, inviting

**White / Off-white**
- Cleanliness, space
- Use for: Text, backgrounds, breathing room
- Emotion: Simple, pure, honest

**Dark Gray/Black**
- Contrast, text
- Use for: Typography, borders, shadows
- Emotion: Professional, readable

### Color Psychology for Your Game:
- **Blue water droplets** = Clean water (good)
- **Gray/black pollutants** = Unsafe water (avoid)
- **Yellow jerry cans** = Solution/action (collect!)
- **White backgrounds** = Cleanliness and clarity

## Iconography & Visual Elements

### Key Icons to Use/Create:
- 💧 **Water droplet** - Clean water, positive
- 🛢️ **Jerry can** (yellow) - charity: water's signature, collection goal
- 🏚️ **Home/community** - Communities being helped
- ✅ **Checkmark** - Milestones achieved
- 🌍 **Globe** - Global impact
- 🚫 **Pollution symbol** - Unsafe water (avoid)
- 📊 **Progress bar** - Building wells/reaching goals

### Typography
- **Title/Headings:** Clean, modern sans-serif (e.g., Arial, Helvetica, Inter)
- **Body text:** Readable, high contrast
- **CTA buttons:** Bold, inviting text

## Branding in Your Game: Key Touchpoints

### 1. **Title Screen**
```
Elements to include:
✓ charity: water logo (top or center)
✓ Game title (your game name)
✓ Tagline connecting to clean water mission
✓ START button (yellow)
✓ Info/About button (optional)
✓ Background: blue + water imagery
```

### 2. **Game Play Screen**
```
Elements to include:
✓ Small charity: water logo (top corner)
✓ Score display (yellow highlight)
✓ Mission counter: "X communities helped"
✓ Collectible icons (yellow jerry cans or water droplets)
✓ Obstacle icons (gray/pollution)
✓ Timer/lives display
✓ Pause button
✓ Background: subtle blue theme, water-themed graphics
```

### 3. **End Game / Milestone Screen**
```
Elements to include:
✓ Celebration message
✓ Final score
✓ Impact statement: "You provided clean water to X people!"
✓ charity: water logo
✓ Link to charitywater.org or donation option
✓ "Play Again" button (yellow)
✓ Social share option (optional)
✓ Background: celebratory blue + gold
```

### 4. **Achievement/Milestone Popups**
```
Elements to include:
✓ Yellow background or border
✓ Star or trophy icon
✓ Achievement text: "Level 2 Unlocked!" or "100 People Helped!"
✓ Small charity: water icon
```

## Brand Integration Ideas by Game Type

### For **Water Quest** (Whack-a-mole):
```
- Jerry cans = collectibles (yellow, branded)
- Pollution drops = obstacles (gray/black)
- Score = represents "clean water access units"
- Milestone message: "You've equipped 1 well!"
- End screen: "You helped 50 people access clean water!"
```

### For **Water Drop** (Falling items):
```
- Blue water droplets = collect (good)
- Black/polluted drops = avoid (bad)
- Container at bottom = community cistern
- Score = gallons collected
- Milestone: "Community well is 50% full!"
- Impact: "This well serves 200 people"
```

### For **Well Builder** (Puzzle/collection):
```
- Grid = land where wells are built
- Yellow icons = water resources collected
- Progress bar = well construction status
- Community cards show real impact numbers
- Victory = well is built, community is helped
- Branding: Before/after images of real charity: water projects
```

### For **Hydration Hero** (Adventure):
```
- Character = the player as a changemaker
- Obstacles = water scarcity challenges
- Communities = points of interest to reach
- Hero's path = journey from scarcity to solution
- Victory = communities now have clean water access
- Ending: Link to charitywater.org to continue the mission
```

### For **Tap for Good** (Clicker):
```
- Big tap area = water droplet or jerry can (yellow)
- Each tap = water unit provided
- Counter = communities helped
- Upgrades = increase impact speed
- Visual progress: "200,000 people now have clean water"
- Final unlock: "Make a real donation" button
```

## Real-World Inspiration

### Design Patterns from charity: water:
- **Bold calls to action** - Clear, yellow, inviting
- **Transparency in numbers** - Show real impact metrics
- **Human stories** - Consider small profile cards of communities helped
- **Global perspective** - Show this is worldwide mission

### Real Elements You Could Reference:

- **Well photos** - Real before/after images
- **Community stories** - Small quotes or names from communities
- **Donation tiers** - "$25 builds 1 well" type messaging

---

## Accessibility & Inclusivity

As you design, consider:
- **Color contrast** - Yellow text needs dark background (or white text on yellow)
- **Color blind friendly** - Don't rely ONLY on color (use icons + color)
- **Font sizing** - Readable on mobile and desktop
- **Clear instructions** - What should the player do?
- **Mobile responsive** - Works on phones, tablets, laptops
