## 📦 Your Complete Package

### Core Game Concept
**Game Name:** AQUA RUSH  
**Tagline:** "Race to Save the World's Water"  
**Genre:** Fast-paced arcade / tap game  
**Platform:** Web (HTML5/CSS/JS)  
**Build Time:** 1-2 weeks  

**Core Mechanic:**  
Players have 60 seconds to tap falling blue water droplets (collect) while avoiding black pollution drops (obstacle). A combo multiplier system (1x → 4x) rewards speed and consistency. Every droplet = 1 person helped.
