## Key Visual Elements to Highlight

### In Your Wireframe, Add Annotations For:

1. **Charity: water Branding**
   - Logo placement (top-left of game screen)
   - Color usage (blue + yellow throughout)
   - Link indicator ("See more at charitywater.org")

2. **User Feedback Areas**
   - Where floating text appears (+10 points)
   - Combo milestone messages
   - Timer warning (turns red)
   - Game over celebration

3. **Interactive Elements**
   - Button states (default, hover, active)
   - Droplet click zones
   - Pause/menu areas

4. **Visual Hierarchy**
   - Score is prominent (largest numbers)
   - Combo is secondary (fire emoji draws attention)
   - Timer is visible but not dominant
   - People helped counter motivates

## Sketch Tips for Hand-Drawing

If drawing by hand:

1. **Use a ruler** for straight lines (borders, button edges)
2. **Draw boxes** for text areas (label them)
3. **Use circles** for droplets (differentiate with shading/color)
4. **Add annotations** with arrows pointing to elements
5. **Include color notes** ("YELLOW BUTTON", "BLUE BACKGROUND")
6. **Show state changes** if possible (combo highlight, timer color change)
7. **Label every element** so developer understands

### Hand-Drawn Example:

```
┌─────────────────────────────┐
│ [Logo]  AQUA RUSH  [Menu]   │ ← Annotation: "Header: blue bg"
├─────────────────────────────┤
│ Score: 2,450  Combo: 2x  32s│ ← Annotation: "Yellow text"
├─────────────────────────────┤
│                             │
│     ●    ●   ●             │ ← Annotation: "Blue droplets (30px)"
│  ●       ●                 │
│     ●         ●     ●      │
│                 ●          │ ← Annotation: "Black = pollution"
│                             │
├─────────────────────────────┤
│ 👥 People Helped: 2,450    │ ← Annotation: "Below game area"
└─────────────────────────────┘

Color key:
🟦 = Blue background
🟨 = Yellow buttons/text
⚫ = Black droplets (pollution)
⚪ = Blue droplets (safe)
```

## What Makes a Strong Wireframe

✅ **Clear labels** - Every element identified  
✅ **Showing all states** - Start, play, end screens  
✅ **Branding visible** - Colors, logo, charity: water elements  
✅ **User flow obvious** - Buttons and interactions clear  
✅ **Easy to photograph** - If hand-drawn, good lighting and contrast  
✅ **High quality image** - Submitted as clear JPG/PNG
